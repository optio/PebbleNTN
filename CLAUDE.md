# Claude Code Instructions

Follow `AGENTS.md` as the primary contract.

Before coding, create or update `docs/IMPLEMENTATION_STATUS.md` with:
- current milestone;
- requirements being implemented;
- commands executed;
- known blockers;
- next atomic task.

Use repository tools rather than manually reproducing generated outputs. Do not ask product questions already resolved by `/spec`. When a library version must be chosen, select the newest stable version compatible with the declared Android baseline, record it in the version catalog, and add a short rationale to the implementation status.

# protocol

Single JSON definition of the AppMessage protocol (keys, events, versions) plus the code
generators that emit the read-only Kotlin and C protocol constants from it.

Source of truth: `examples/protocol-definition.json` (mirrors `spec/200-architecture/Protocol.md`).
Generated protocol files are **never** hand-edited (AGENTS.md rule 5).

Implemented in **M1 — Domain model and protocol**. Placeholder in M0.
